# Cipher v1.0
A Caesar cipher CLI tool written in C++.

## Usage
```bash
g++ cipher.cpp -o cipher
sudo ./cipher
```

## Features
- Encrypts strings using a shift of 3
- Option to decrypt after encryption
- Handles upper/lowercase, leaves symbols and numbers unchanged
